# Joseph Resch Website

This version is configured for the `alshedivat/al-folio` remote theme and keeps the site content aligned to the resume.

## Deploying on GitHub Pages

1. Push these files to the `josephresch.github.io` repository.
2. In GitHub Pages settings, deploy from the main branch if needed.
3. GitHub Pages should build the site using the remote theme.

If GitHub Pages does not build immediately, check that the repository is enabled for Pages and that Actions are allowed.
